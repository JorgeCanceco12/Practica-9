Bases de Datos

Las bases de datos son sistemas que permiten almacenar, gestionar y recuperar información de manera eficiente. Se dividen principalmente en:

Tipos de Bases de Datos

  • Relacionales: Utilizan tablas para organizar la información. Ejemplo: MySQL, PostgreSQL.
  • No relacionales: A menudo se utilizan para grandes volúmenes de datos no estructurados. Ejemplo: MongoDB, Couchbase.

Consultas SQL

El lenguaje de consulta estructurado (SQL) es fundamental para interactuar con bases de datos relacionales. Algunos ejemplos de consultas son:

  • SELECT * FROM usuarios; - Selecciona todos los registros de la tabla usuarios.
  • INSERT INTO productos (nombre, precio) VALUES ('Café', 5.00); - Inserta un nuevo producto en la tabla.
